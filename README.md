# we-spot
A modest Spot It! / Dobble clone written with Angular and Ionic Framework.

# Quick Start

1. Clone the repo
1. run `npm install`
1. run `npm start`

# High Level Features I Hope to Include
- Training mode
- Multiple levels of difficulty
- "Offline" multiplayer
- "Realtime" multiplayer
- Customizable images
- Google Play support
- PWA deployment and support
